Welcome to Version 1.0 of our API !


You can access our data for free, compliments of PetFinder.my


You may download the SQLite file from here


Start your API session by requesting the Adoption data attributes so you know what is available

/api/v1.0/Adoptions

["Age","Breed1","Breed2","Gender","Color1","Color2","Color3","MaturitySize","FurLength","Vaccinated","Dewormed","Sterilized","Health","Quantity","Fee","State","RescuerID","VideoAmt","Description","PetID","PhotoAmt","AdoptionSpeed"]

Each attribute is added to the API path, followed by the sample number you want to retrieve

For example, to retrieve breed number 1:

/api/v1.0/breed/1

["Pet Breed:",[1],["Affenpinscher"]]

To show all breeds:

/api/v1.0/Breeds


The first number in the list is the total number of samples (i.e. there are 307 breeds broken into dogs and cats)

You can try the same for each attribute by Capitalizing the attribute name and making it plural (add an 's')

/api/v1.0/Colors

["Pet Colors:","7",["Black","Brown","Golden","Yellow","Cream","Gray","White"]]
